Data Privacy Compliance Checklist

Last updated: Jan 2025

As per the Data Protection Act (2019), integrations utilizing PayMo APIs must ensure:

  • Explicit Consent: Utilize the /v1/privacy/consent/update endpoint before capturing marketing parameters.
  • Right to Erasure: Call the /v1/privacy/data/erase endpoint when a user closes their account. Note: Financial records are exempt for 7 years (AML laws).
  • Data Minimization: Only query required scopes. PayMo OAuth tokens will reject out-of-scope requests automatically.
Did you know? Integrating the DSAR API automates user data requests, saving your support team hours of manual compilation.
