GraphQL API Support

Category: Core Architecture Under Review

89

Use Case: Our mobile application requires data from 4 different REST endpoints to build the dashboard (wallet, invoices, stats, profile). This causes over-fetching and impacts performance on slower 3G networks. A GraphQL endpoint would solve this.

Official Response
SD
Sarah D. (PM)

Thanks for the feedback! We are aware of the payload size issues for mobile integrators. GraphQL support is officially moving to our "Planned" roadmap for v3.0, expected in Beta late next year.
